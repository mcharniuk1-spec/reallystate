# SQL helper scripts

Run against the **same** database URL you use for `make db-init` (local: `postgresql://bgrealestate:bgrealestate@localhost:5432/bgrealestate`).

## From Docker (no local `psql` required)

```bash
docker compose exec -T postgres psql -U bgrealestate -d bgrealestate < sql/helpers/01_list_public_tables.sql
docker compose exec -T postgres psql -U bgrealestate -d bgrealestate < sql/helpers/02_listing_media_sample.sql
```

## Files

| File | Purpose |
|------|---------|
| `01_list_public_tables.sql` | List `public` tables + rough row estimates |
| `02_listing_media_sample.sql` | Join `listing_media` → `canonical_listing` for inspection |

DDL for all tables is generated by Alembic from `sql/schema.sql` (see `migrations/versions/20260407_0001_initial_schema.py`).
