# Agent Journeys (5 roles)

This project uses 5 specialist agents plus the lead agent.

Each specialist agent must keep a **journey log** under `docs/agents/<agent>/JOURNEY.md`:

- append every task executed (what/why)
- link to changed files
- note verification (commands/tests)
- leave “review comments” after execution (what to improve next time)

Agents:

- `debugger`
- `backend_developer` (data engineer / backend structure)
- `ux_ui_designer` (frontend)
- `scraper_1` (marketplace websites)
- `scraper_sm` (social/media overlays; consent-gated)
